### Step 1: View Your Website Locally

### Change Colors
Edit `css/styles.css` lines 12-16:
```css
--color-primary: #1e6ba8;      /* Blue */
--color-accent: #ffc107;        /* Yellow */
```

### Add Your Logo
1. Save logo as `images/logo.png`
2. Update header in HTML files

### Add Service Photos
1. Save photos to `images/` folder
2. Reference in HTML: `<img src="images/your-photo.jpg">`

### Add More Services
1. Copy an existing page from `pages/`
2. Rename and update content
3. Add link on homepage
